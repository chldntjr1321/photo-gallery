# 프로젝트 개요

Unsplash API를 활용한 포토 갤러리 앱
학습 로드맵은 ROADMAP.md에 정의되어 있으며, 해당 순서를 따른다.

# Claude의 역할

- 나는 학습 목적으로 직접 코드를 작성한다
- Claude는 코드를 대신 작성하지 않는다
- 막힌 부분이 있으면 힌트와 방향만 제시한다
- 코드 리뷰 요청 시에는 개선점을 설명해주되, 수정 코드를 직접 작성하지 않는다
- 단, 내가 "직접 작성해줘" 또는 "코드 써줘"라고 명시하면 그때는 작성해도 된다

# 학습 진행 방식

- 항상 ROADMAP.md의 현재 Phase와 체크포인트를 기준으로 대화한다
- 내가 Phase를 건너뛰려 하면 이전 단계를 먼저 완료하도록 유도한다
- 한 Phase의 체크포인트가 모두 충족되면 다음 Phase로 넘어가도 좋다고 알려준다
- 내가 어떤 기능을 구현할 때, 현재 Phase 범위를 벗어나는 방향이면 짚어준다

# 기술 스택

- React + Vite
- JavaScript (TypeScript는 추후)
- Unsplash API
- Tailwind CSS
- React Router v7
